\begin{abstract}
Federated learning is attractive for industrial Internet of Things (IIoT) intrusion detection because edge gateways observe sensitive and heterogeneous telemetry that should not be centralized. However, federated learning alone does not provide a formal privacy guarantee, and private fine-tuning of language models often suffers from high communication cost and severe utility degradation under client-side differential privacy (DP). We propose \method, an auditable private federated adaptation framework for edge IIoT security assistants. \method constructs fixed adapter bases only from the public spectral structure of a frozen 2B-parameter Qwen3.5 backbone, trains tiny layer-wise core matrices, and releases clipped and noised core updates under client-level DP. To reduce the performance loss caused by client-side release DP, \method couples public spectrum-driven rank allocation, layer-adaptive clipping and noise allocation, local residual personalization, and privacy-cost-free server post-processing. We evaluate the framework on \dataset using a mainstream 80/10/10 stratified closed-set protocol and simulated federated client partitions on one server. \end{abstract}
